*VaultX*
VaultX is a cross-platform Flutter application that helps users store, organize, and manage important information in a structured and user-friendly way. Built with Flutter, the app focuses on simplicity, performance, and a clean modern design.

Features
Clean and responsive UI
Secure data organization
Fast and intuitive navigation
Cross-platform support (Android & iOS)
Modern Flutter architecture
Easy-to-use interface
Tech Stack
Flutter
Dart
Firebase
Material Design
